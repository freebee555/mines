# 💣 扫雷 — 网页版

纯 HTML/CSS/JS 单文件扫雷游戏，零依赖，打开即玩。适配桌面端和移动端。

## 🎮 在线试玩

> 直接打开 `index.html` 即可在浏览器中运行。

## ✨ 功能特性

- **经典 Windows 扫雷体验** — 还原经典灰色立体按钮风格
- **三种难度** — 简单 9×9 / 中等 16×16 / 困难 16×30
- **自适应棋盘** — 根据屏幕尺寸自动缩放格子大小，手机/平板/桌面完美适配
- **双模式切换** — ⛏ 挖掘模式 / 🚩 插旗模式，一键切换
- **触屏支持** — 移动端长按插旗，双击数字快速翻开相邻格子
- **胜利特效** — 通关后彩色纸屑飘落动画 🎊
- **最佳成绩** — 各难度自动记录最佳时间与胜率（localStorage）
- **暂停/继续** — 切到后台自动暂停，回到标签页继续
- **声音效果** — Web Audio 合成翻开/插旗/踩雷/胜利音效，可一键静音
- **深色模式** — 自动跟随系统的 `prefers-color-scheme`
- **键盘操作** — 方向键移动、Space/Enter 翻开、F 插旗、Esc 重开
- **可访问性** — `<button>` 语义化结构、`aria-label` 状态描述、屏幕阅读器友好
- **零依赖** — 不依赖任何框架或库，单文件即可运行

## 🚀 使用方式

```bash
# 方式一：直接在浏览器打开
open index.html

# 方式二：用任意 HTTP 服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 💾 数据本地存储

游戏会在 `localStorage` 中保存：
- `mines_stats_v1`：每个难度的最佳时间、总局数、胜场数
- `mines_sfx`：声音开关偏好（`1` / `0`）

清空浏览器存储即可重置。

## 🛠 技术实现

- **HTML5** — 单页面结构，语义化 `<button>` 标签
- **CSS3** — Grid 布局、CSS 变量（含 `prefers-color-scheme` 暗色主题）、响应式 `clamp()`、`@keyframes` 动画
- **JavaScript (ES5+)** — IIFE 封装，Web Audio API 合成音效，localStorage 持久化，visibilitychange 自动暂停
- **可访问性** — `role="grid"`、`aria-label` 状态描述、键盘全功能操作
