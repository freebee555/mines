# 💣 扫雷 — 网页版

纯 HTML/CSS/JS 单文件扫雷游戏，零依赖，打开即玩。适配桌面端和移动端。

## 🎮 在线试玩

> 直接打开 `index.html` 即可在浏览器中运行。

## ✨ 功能特性

- **经典 Windows 扫雷体验** — 还原经典灰色立体按钮风格
- **三种难度** — 简单 9×9 / 中等 16×16 / 困难 16×30
- **自适应棋盘** — 根据屏幕尺寸自动缩放格子大小，手机/平板/桌面完美适配
- **双模式切换** — ⛏ 挖掘模式 / 🚩 插旗模式，一键切换
- **触屏支持** — 移动端长按插旗，双击数字快速翻开相邻格子
- **胜利特效** — 通关后彩色纸屑飘落动画 🎊
- **零依赖** — 不依赖任何框架或库，单文件即可运行

## 🕹 操作方式

| 操作 | 桌面端 | 移动端 |
|------|--------|--------|
| 翻开格子 | 左键点击 | 点击（挖掘模式） |
| 插旗/取消 | 右键点击 | 长按 / 点击（插旗模式） |
| 快速翻开 | 双击数字格子 | 点击数字格子 |
| 切换模式 | 点击 ⛏/🚩 按钮 | 点击 ⛏/🚩 按钮 |
| 重新开始 | 点击 🙂 笑脸 | 点击 🙂 笑脸 |

## 🚀 使用方式

```bash
# 方式一：直接在浏览器打开
open index.html

# 方式二：用任意 HTTP 服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 🛠 技术实现

- **HTML5** — 单页面结构，语义化标签
- **CSS3** — Grid 布局、CSS 变量、响应式 `clamp()`、`@keyframes` 动画
- **JavaScript (ES6)** — IIFE 封装，全局仅暴露 3 个函数，无全局变量污染
